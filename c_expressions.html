3.What would be the output of the following program ? 
#include<stdio.h>
int main()
{
    int i=-3, j=2, k=0, m;
    m = ++i && ++j && ++k;
    printf("%d, %d, %d, %d\n", i, j, k, m);
    return 0;
}

Explanation

Option A is the correct answer.
m = ++i && ++j && ++k; becomes m = -2 && 3 && 1; becomes m = TRUE && TRUE; Hence this statement becomes TRUE. So it returns '1'(one). Hence m=1.

4.what would be the output of the following program ? 
#include<stdio.h>
int main()
{
    int i=-3, j=2, k=0, m;
    m = ++i || ++j && ++k;
    printf("%d, %d, %d, %d\n", i, j, k, m);
    return 0;
}

Explanation

Option D is the correct answer.
m = ++i || ++j && ++k; here (++j && ++k;) this code will not get executed because ++i has non-zero value. becomes m = -2 || ++j && ++k; becomes m = TRUE || ++j && ++k; Hence this statement becomes TRUE. So it returns '1'(one). Hence m=1.

7.What will be the output of the program ? 
#include<stdio.h>
int main()
{
    int x=4, y, z;
    y = --x;
    z = x--;
    printf("%d, %d, %d\n", x, y, z);
    return 0;
}

Explanation

Option D is the correct answer.
y = --x; becomes y = 3; because (--x) is pre-decrement operator. z = x--; becomes z = 3;. In the next step variable x becomes 2, because (x--) is post-decrement operator. Hence it prints "2, 3, 3".
