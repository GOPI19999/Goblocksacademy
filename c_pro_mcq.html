int main() 
{
   extern int a; 
   printf("%d",a);
}

int a=2;

Explanation

Option A is the correct answer.
extern int i is a declaration and int i=2 is the definition .

3.What would be the output of the following program ? 
int main() 
{
   display();
}

void display() 
{
   printf("gopi");
}

Explanation

Option C is the correct answer.
Here display() is called before it is defined.In such cases the compiler assumes that the function display() is declared as int display() but in function definition we define the return type as void.
