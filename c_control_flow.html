3.What would be the output of the following program ? 
int main() 
{
  int i=2,j=3; 
  switch(i) 
  {
  
     case 1: 
       printf("%d",1); 
     case 2 : 
        printf("%d",2);
     case j : 
        printf("%d",j);
     default : 
        printf("%d",i);

  }
}

Explanation

Option C is the correct answer.
Case label should be int constant .(identifier not allowed in case label but allowed in switch condition eg switch(i) ).
